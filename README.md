# DiscordStatus

Version: 1.0.0

Программа создана для смены статуса в программе Discord.

**Первоначальная настройка:**
1. Запустить ***install.bat***
2. Указываем свой токен в ***cfg.json***

Работа с приложением осуществляется через командную строку или bat-файлы. (start.bat в качестве примера установит статус играет в Dota 2)

**Команды:**
  1. ***game*** - Установит статус играет в %название игры%. 
	Пример команды: ***node ds game --g="Dota 2"***
  2. ***listen*** - Установит статус слушает %название песня%. 
	Пример команды: ***node ds listen --g="Dota 2"***
  3. ***watch*** - Установит статус смотрит %название видео%. 
	Пример команды: ***node ds watch --g="Dota 2"***
  4. ***stream*** - Установит статус стримит %название игры% с url-адресом на ваш Twitch. 
	Пример команды: ***node ds watch --g="Dota 2" --url="https://www.twitch.tv/"***
